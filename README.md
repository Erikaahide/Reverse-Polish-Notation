 **Reverse Polish Notation**  
Hi, this project was created using plain JavaScript (no frameworks).This showcases key concepts as stack (LIFO: Last In, First Out), tokenizaiton, functional thinkig, error handling, also *edge cases* as empty input, invalid token, insufficient operands, division by zero.

## Tech Stack
- HTML
- JavaScript (ES6+)
- CSS3 

 **How it works** 
 Instead of writing expressions like: (3 + 4) * 2     RPN expresses it as: 3 4 + 2 *

 RPN is a mathematical notation where operators follow their operands. It removes the need for parentheses and makes expression evaluation straightforward using a stack.
